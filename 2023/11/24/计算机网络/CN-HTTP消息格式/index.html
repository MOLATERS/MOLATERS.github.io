[[2023-09-17]] # HTTP请求消息

HTTP有两类消息

  • 请求消息
  • 响应消息

请求消息

ASCII：人直接可读

HTTP请求消息的通用格式

上传输入的方法

POST方法

  • 网页需要填写表格
  • 在请求消息的消息体（entity body）中上传客户端的输入

URL方法

  • 使用GET方法
  • 输入信息通过request行的URL字段上传

方法的类型

HTTP/1.0

  • GET/POST/[[CourseNotes/计算机网络/CN-HEAD]]

HTTP/1.1

  • GET,POST,HEAD
  • [[CourseNotes/计算机网络/CN-PUT]]
  • [[CourseNotes/计算机网络/CN-DELETE]]

HTTP的响应消息

HTTP响应状态代码

  • 响应消息是第一行

  • 示例

    • 200 OK
    • 301 Moved Permanently
    • 400 Bad Requeast
    • 404 Not Found
    • 505 HTTP Version Not Supported
